hexo+github踩过的坑实录

流程和坑
第一步，先准备安装环境
1， 安装git，
用于安装hexo，并部署到github

2， 安装Node.js

3， 下载并安装hexo
右击，进入git bash here，输入npm install -g hexo，等待下载并安装完成

4，github仓库和page创建
注册账号，创建仓库，仓库命名：
创建gitpage
先前往github注册账号，创建仓库注意命名规则：
第二步，在本地生成代码
1， 初始化
新建文件夹hexo，用于存放代码；
在文件夹空白处，右击打开git，输入$ hexo init；
这时候在hexo中已经看到已经生成了代码。
2， 安装依赖包
npm install，具体什么是依赖包，不知道。
3，本地部署
hexo generate，用于生成文章页代码；
hexo server，用于部署到本地；
4，本地测试
默认端口4000，访问即可 http://localhost:4000/；
然而福晰阅读器会占用这个端口，需要修改默认端口。
5，修改端口
hexo中找到_config.yml，打开，添加
server:
port: 5000
compress: true
header: true
保存；
hexo s，就可以访问下面的地址；
http://localhost:5000/
第三步，将本地hexo部署到github
